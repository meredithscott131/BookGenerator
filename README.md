## Features
- Horizontal and vertical orientation toggling
- Aspect ratio customization
- Depth (thickness) customization
- Rotation customization
- Color palette
- Randomization adjustment

## Setup
1. Download the most recent release of this repository
2. Launch Houdini and navigate to `File` → `Import` → `Houdini Digital Asset`
3. Select the downloaded `BookGenerator.hdanc` file
4. Press `Install and Create`

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[More information on Houdini Digital Assets](https://www.sidefx.com/docs/houdini/assets/install.html)

### Files Included
- BookGenerator.hdanc
- BookGenerator.hipnc

### Requirements
- Houdini 18+
- Beginner knowledge of Houdini
